# Little-Pomodoro - 小番茄，专注 30 分钟
免费番茄钟计时器，番茄时钟在线网页，专注工作学习<br>
<br>
关于：<br>
左为轮次 · 右为分钟 · 每轮 30 分钟<br>
**只显示30分钟**，不支持自定义时间，因为休息就该好好休息，30分钟更像完整的数字<br>
一天**专注完成3个任务**就已经很棒了，没有任务备注跟细分<br>
点击任务左侧图标可**高亮当前任务**，再点击"取消"<br>
专注30分钟，**不支持暂停**，每次点击都是重新开始<br>
**没有日/周记录报表**，任务本身就是最好的正反馈，尽量把视线放在任务本身哦<br>
只设置了**开始提示音**，没有环境音，尽量减少干扰<br>
